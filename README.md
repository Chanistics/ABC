# ABC: Advanced Bench Calculator

Bench-side biomedical calculator for molecular weight, molarity, concentration conversion, dilution, buffer recipes, and central dogma sequence utilities.

## GitHub Pages

This app is a static site. Deploy the repository root with GitHub Pages:

1. Open the repository on GitHub.
2. Go to `Settings` -> `Pages`.
3. Set `Source` to `Deploy from a branch`.
4. Select branch `main` and folder `/root`.
5. Save.

The deployed URL will look like:

```text
https://<github-username>.github.io/<repository-name>/
```

## Files

- `index.html`: app shell and UI
- `styles.css`: responsive styling
- `app.js`: calculator logic
- `manifest.webmanifest`: installable PWA metadata
- `service-worker.js`: offline app shell cache
- `icons/`: PWA icons
